# Developer keys

**Developer keys are planned to be phased out in the future**

## What are developer keys?

Developer keys are keys stored in the bots database, they are only given out to trusted developers

## What do developer keys do?

Developer keys allow anyone with the key to modify certain things about the bot,

For example a developer key means you can edit parts of the database.

Developer keys are meant for quick fixes or quickly helping a user
Developer keys are often changed quickly after use

## How do I get my own developer key?

Developer keys are not just given out, they require years of trust

Their is no single way to get a developer key, they are only given out in rare cases

## What developer keys DO NOT DO

Developer keys DO NOT:   
    Allow people to attack your server   
    Give people full control over the bot   
    Give access to personal infomation    
    Give access to directly modify Autonomy's code
